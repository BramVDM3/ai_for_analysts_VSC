# Library AI for Analysts

Starter project for practicing an AI-supported analyst workflow in VS Code. The repository combines raw analysis material, a small FastAPI backend, and a frontend so you can inspect the current system and create evidence-based requirements.

## Start Here

- [Get Started](GET_STARTED.md): setup instructions for VS Code, AI tooling, and optional local app setup.
- [Exercise Explanation](EXCERSISE_EXPLANATION.md): the analysis assignment, expected outputs, and final presentation goal.

## Repository Map

- `analysis/`: source materials, models, requirements, epics, and stories.
- `backend/`: Python/FastAPI application and backend tests.
- `frontend/`: Vite frontend for the library app.
- `mock-database/`: sample CSV data used by the exercise.

## Main Goal

Use the raw materials and current application as evidence to:

- understand the existing library workflow
- identify stakeholder needs, frustrations, and conflicts
- create as-is and to-be models
- write epics, stories, acceptance criteria, and test scenarios
- prepare a concise board presentation
